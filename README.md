This is a readme for Subsurface Extraction
Made for a game for gamjam campfire prague in godot 4.6

Known bugs:
- Sometimes sound cuts off
- There may be onter undiscoverd bugs - please report

Tutorial:
- A/D to move, W/SPACE to jump
- Oxygen replanishes near station on top of first iland
- You can smelt and sell and buy near station
- Goal of the gae is to pine as much as possible and amake as much money as possible
- 1/2/3 for different building blocks
- There is partial controler support but as of now there isnt way to mine or place

How to play:
- You can play the web version at: https://shift7-cz.itch.io/subsurface-extraction
- Linux/Windows bulds are inside the build folder (We recomend the web version thouth - its more pollished)
- For other platforms you can build it yourself by using Godot 4.6
